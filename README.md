## How to run it

### Start server side

Make sure you have mongoDB installed locally

cd /server
run `npm start` or `yarn start`

### Start clien side

cd /client
run `npm run dev-server`

### Features

- [✅] Reservation CRUD
- [✅] User CRUD
- [✅] JWT Authentication
- [✅] Role based authorization
- [✅] Restful API
- [✅] Graphql server
- [✅] SPA client
